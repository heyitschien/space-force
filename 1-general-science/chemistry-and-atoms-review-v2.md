Most Chemistry & Atoms questions test whether you can quickly identify:

2. **Atomic identity rules** (atomic number, atom definition)
3. **Matter classification language** (element, molecule, compound, mixture)
4. **Core chemical properties** (acids, oxidation)

If you know definitions + contrasts, this section becomes predictable.

## 2) Must-know facts (high yield)

### Atomic structure

- **Electron** has a **negative** charge.
- **Proton** has a positive charge and defines element identity.
- **Neutron** is neutral.
- **Atomic number** = number of **protons**.
- **Atom** = smallest part of an element that still behaves like that element.

### Matter vocabulary

- **Element**: one type of atom.
- **Molecule**: atoms bonded together.
- **Compound**: two or more different elements chemically bonded.
- **Mixture**: substances physically combined, not chemically bonded.

### Core chemistry patterns

- **Oxidation** (test framing): loss/removal of electrons.
- **Acids**: pH below 7 and typically sour taste.
- **Chlorophyll** is the pigment used in photosynthesis.

## 3) Definition -> Contrast -> Example map

Use this pattern for every concept:

### Atomic Number

- Definition: Number of protons in the nucleus.
- Contrast: Not number of electrons.
- Example: 8 protons means oxygen.

### Atom

- Definition: Smallest unit that still acts like an element.
- Contrast: Not molecule/compound/mixture.
- Example: One oxygen atom still has oxygen identity.

### Compound vs Mixture

- Definition (compound): Chemical bond between different elements.
- Contrast (mixture): Physical blend, no new chemical identity.
- Example: Water is a compound; salad is a mixture.

### Oxidation

- Definition: Electron loss.
- Contrast: Not electron gain.
- Example: If species loses electrons in a reaction, it is oxidized.

## 4) Trap answers you should expect

### Trap 1: particle swap

- Atomic number choices include electron count to trick you.
- Fix: atomic number always maps to proton count.

### Trap 2: atom vs element wording

- Question asks smallest part that still acts like element.
- Correct keyword: atom.

### Trap 3: unit/condition mismatch

- Acid questions may mix pH facts with taste distractors.
- Fix: acids = pH < 7 and sour.

### Trap 4: near-true category choices

- Molecule/compound/mixture can all look plausible.
- Fix: ask “chemical bond or physical blend?”
