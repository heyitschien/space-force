# ⚡ LEVEL 2 — RATE × QUANTITY (MASTER SYSTEM)

## 🔥 CORE IDEA (Lock This In)

👉 Every ASVAB rate problem is THIS:

\text{Total} = \text{Rate} \times \text{Quantity}

---

## 🧠 THE 3 FORMS (You MUST be fluent)

1. **Total = Rate × Quantity**
2. **Rate = Total ÷ Quantity**
3. **Quantity = Total ÷ Rate**

👉 This is your **triangle**:

```
        Total
      /       \
   Rate     Quantity
```

Cover one → divide
Want one → divide
Have both → multiply
