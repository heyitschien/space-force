### Addition: Combining Quantities

<div style="background: #e8f5e9; padding: 15px; border-radius: 8px; border-left: 5px solid #4caf50; margin: 15px 0;">
  <h3 style="color: #2e7d32; margin-top: 0;">ğŸŸ¢ What Addition Means</h3>

**Key Words:** sum, total, combined, together, plus, add

#### Example 1: Simple Addition

<div style="background: #e3f2fd; padding: 15px; border-radius: 8px; border-left: 5px solid #2196f3; margin: 15px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0;">Sarah has 23 books. She buys 17 more books. How many books does she have in total?</p>
</div>

**Step-by-Step Solution:**

**Step 1:** Identify what you're adding
- Sarah's current books: 23
- New books: 17
- We want the total

**Step 2:** Set up the addition
```
  23
+ 17
----
```

**Step 3:** Add the ones place (right column)
- 3 + 7 = 10
- Write 0, carry 1 to tens place

**Step 4:** Add the tens place (left column)
- 2 + 1 + 1 (carried) = 4
- Write 4

**Step 5:** Write the answer
```
  23
+ 17
----
  40
```

**Answer:** 40 books

**Why This Works:** Addition combines quantities. We add the ones place first, then the tens place, carrying when needed.
