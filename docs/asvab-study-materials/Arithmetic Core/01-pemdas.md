<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <p style="margin: 0;"><strong>Big idea:</strong> PEMDAS is not just a mnemonic—it's a <strong>priority system</strong>.</p>
  <p style="margin: 10px 0 0 0;"><strong>Most common mistake:</strong> thinking “MD” means multiply before divide. It does <strong>not</strong>. It means <strong>do multiplication/division left-to-right</strong>.</p>

## 1) The Fundamental Rule (PEMDAS)

<div style="background: #f3e5f5; padding: 16px; border-radius: 10px; border-left: 6px solid #9c27b0; margin: 18px 0;">
  <h3 style="color: #7b1fa2; margin-top: 0;">🟣 PEMDAS Priority</h3>
  <p style="margin: 10px 0; font-size: 1.15em;"><strong>P</strong>arentheses / Grouping → <strong>E</strong>xponents → <strong>M</strong>ultiply / <strong>D</strong>ivide → <strong>A</strong>dd / <strong>S</strong>ubtract</p>
  <ul style="margin: 10px 0 0 0; line-height: 1.8;">
    <li><strong>MD:</strong> same “priority tier” → go <strong>left to right</strong></li>
    <li><strong>AS:</strong> same “priority tier” → go <strong>left to right</strong></li>
    <li><strong>Grouping symbols include:</strong> ( ), [ ], { }, and the <strong>fraction bar</strong></li>
  </ul>
  <p style="margin: 10px 0 0 0;"><strong>Memory tip:</strong> “Please Excuse My Dear Aunt Sally.”</p>
</div>

### Worked Example 1 (warm-up)

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">
  <h4 style="color: #e65100; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$8 + 3 \times 2$$</p>
</div>

**Step-by-step (showing work):**
1. Multiplication first: $$3 \times 2 = 6$$
2. Then addition: $$8 + 6 = 14$$

**Answer:** 14

## 3) Exponents (E) Come Before Multiply/Divide

<div style="background: #e8f5e9; padding: 16px; border-radius: 10px; border-left: 6px solid #4caf50; margin: 18px 0;">

  <p style="margin: 10px 0;"><strong>Exponents</strong> mean repeated multiplication: $$2^3 = 2 \times 2 \times 2$$</p>
  <p style="margin: 10px 0 0 0;"><strong>Common trap:</strong> $$-3^2 \text{ means } -(3^2) \text{ unless parentheses are used.}$$</p>

## 4) Multiply/Divide: Same Priority, Go Left-to-Right

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h3 style="color: #1976d2; margin-top: 0;">🔵 Fundamental Principle</h3>
  <p style="margin: 10px 0;">When you see a chain of × and ÷, <strong>do them in order from left to right</strong>.</p>
</div>

### Worked Example 4 (the classic trap)

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">

  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$8 \div 2 \times 4$$</p>
</div>

## 6) Fractions + PEMDAS (the “fundamental principle” you asked for)

<div style="background: #ffebee; padding: 16px; border-radius: 10px; border-left: 6px solid #f44336; margin: 18px 0;">
  <h3 style="color: #c62828; margin-top: 0;">🔴 Fundamental Principle: Fraction Bar = Grouping</h3>
  <p style="margin: 10px 0;"><strong>A fraction bar means “numerator ÷ denominator.”</strong></p>
  <p style="margin: 10px 0 0 0;">That bar also means: do the entire numerator and entire denominator first (like parentheses), then divide.</p>
</div>

### Worked Example 6 (fraction with multiplication inside)

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$\dfrac{6 + 2 \times 5}{4}$$</p>
</div>

**Step-by-step (showing work):**
1. Work inside the numerator using PEMDAS: $$2 \times 5 = 10$$
2. Numerator becomes: $$6 + 10 = 16$$
3. Divide by the denominator: $$\dfrac{16}{4} = 4$$

**Answer:** 4

### Worked Example 7 (fractions + MD left-to-right)

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$\dfrac{3}{4} \times 8 \div 2$$</p>
</div>

**Step-by-step (showing work):**
1. Do × and ÷ left-to-right:
2. $$\dfrac{3}{4} \times 8 = \dfrac{3 \times 8}{4} = \dfrac{24}{4} = 6$$
3. Then divide: $$6 \div 2 = 3$$

**Answer:** 3

### Worked Example 8 (dividing by a fraction)

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$\dfrac{3}{5} \div \dfrac{1}{10} + 2$$</p>
</div>

**Step-by-step (showing work):**
1. Division of fractions: multiply by the reciprocal  
   $$\dfrac{3}{5} \div \dfrac{1}{10} = \dfrac{3}{5} \times \dfrac{10}{1}$$
2. Multiply: $$\dfrac{3 \times 10}{5} = \dfrac{30}{5} = 6$$
3. Now add: $$6 + 2 = 8$$

**Answer:** 8

## 7) Complex Fractions (fraction-on-fraction)

<div style="background: #e8f5e9; padding: 16px; border-radius: 10px; border-left: 6px solid #4caf50; margin: 18px 0;">
  <h3 style="color: #2e7d32; margin-top: 0;">🟢 Fundamental Principle</h3>
  <p style="margin: 10px 0;">A “complex fraction” means <strong>(top) ÷ (bottom)</strong>. Treat the entire top and entire bottom as grouped.</p>
  <p style="margin: 10px 0 0 0;"><strong>Best move:</strong> rewrite as multiplication by the reciprocal.</p>
</div>

### Worked Example 9

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$\dfrac{\dfrac{1}{2}}{\dfrac{3}{4}}$$</p>
</div>

**Step-by-step (showing work):**
1. Rewrite as division: $$\dfrac{1}{2} \div \dfrac{3}{4}$$
2. Multiply by reciprocal: $$\dfrac{1}{2} \times \dfrac{4}{3}$$
3. Multiply: $$\dfrac{4}{6} = \dfrac{2}{3}$$

**Answer:** $$\dfrac{2}{3}$$
