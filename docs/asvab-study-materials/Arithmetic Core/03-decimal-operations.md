## 1) Understanding Decimal Place Value

<div style="background: #f3e5f5; padding: 16px; border-radius: 10px; border-left: 6px solid #9c27b0; margin: 18px 0;">
  <h3 style="color: #7b1fa2; margin-top: 0;">🟣 Decimal Place Value</h3>
  <p style="margin: 10px 0;">Every digit in a decimal has a place value based on its position relative to the decimal point.</p>
  <p style="margin: 10px 0; font-size: 1.1em;"><strong>Example:</strong> In 3.456</p>
  <ul style="margin: 10px 0 0 0; line-height: 1.8;">
    <li><strong>3</strong> = ones place</li>
    <li><strong>4</strong> = tenths place (0.4)</li>
    <li><strong>5</strong> = hundredths place (0.05)</li>
    <li><strong>6</strong> = thousandths place (0.006)</li>
  </ul>
  <p style="margin: 10px 0 0 0;"><strong>Memory tip:</strong> Read decimals as "three and four hundred fifty-six thousandths" or "three point four five six."</p>
</div>

## 2) Adding Decimals

<div style="background: #e8f5e9; padding: 16px; border-radius: 10px; border-left: 6px solid #4caf50; margin: 18px 0;">
  <h3 style="color: #2e7d32; margin-top: 0;">🟢 Fundamental Principle: Line Up the Decimal Points</h3>
  <p style="margin: 10px 0;"><strong>Rule:</strong> When adding decimals, always align the decimal points vertically. Then add column by column, just like whole numbers.</p>
  <p style="margin: 10px 0 0 0;"><strong>Key steps:</strong></p>
  <ol style="margin: 10px 0 0 0; line-height: 1.8;">
    <li>Write the numbers vertically with decimal points aligned</li>
    <li>Add zeros to the right if needed to make all numbers have the same number of decimal places</li>
    <li>Add column by column from right to left</li>
    <li>Carry when needed (just like whole numbers)</li>
    <li>Place the decimal point in the answer directly below the decimal points in the problem</li>
  </ol>
</div>

### Worked Example 1: Simple Addition

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">
  <h4 style="color: #e65100; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$3.45 + 2.17$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Write vertically with decimal points aligned
```
  3.45
+ 2.17
------
```

**Step 2:** Add column by column from right to left
- Hundredths: 5 + 7 = 12 → write 2, carry 1
- Tenths: 4 + 1 + 1 (carried) = 6 → write 6
- Ones: 3 + 2 = 5 → write 5

**Step 3:** Place decimal point in answer
```
  3.45
+ 2.17
------
  5.62
```

**Answer:** 5.62

### Worked Example 2: Different Decimal Places

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">
  <h4 style="color: #e65100; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$7.3 + 4.25 + 0.156$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Write vertically with decimal points aligned. Add zeros to make all numbers have 3 decimal places
```
  7.300
  4.250
+ 0.156
------
```

**Step 2:** Add column by column
- Thousandths: 0 + 0 + 6 = 6
- Hundredths: 0 + 5 + 5 = 10 → write 0, carry 1
- Tenths: 3 + 2 + 1 + 1 (carried) = 7 → write 7
- Ones: 7 + 4 + 0 = 11 → write 1, carry 1
- Tens: 0 + 0 + 0 + 1 (carried) = 1 → write 1

**Step 3:** Place decimal point
```
  7.300
  4.250
+ 0.156
------
 11.706
```

**Answer:** 11.706

## 3) Subtracting Decimals

<div style="background: #ffebee; padding: 16px; border-radius: 10px; border-left: 6px solid #f44336; margin: 18px 0;">
  <h3 style="color: #c62828; margin-top: 0;">🔴 Fundamental Principle: Same as Addition</h3>
  <p style="margin: 10px 0;"><strong>Rule:</strong> Line up the decimal points vertically. Subtract column by column, borrowing when needed.</p>
  <p style="margin: 10px 0 0 0;"><strong>Key steps:</strong></p>
  <ol style="margin: 10px 0 0 0; line-height: 1.8;">
    <li>Write the numbers vertically with decimal points aligned</li>
    <li>Add zeros to the right if needed</li>
    <li>Subtract column by column from right to left</li>
    <li>Borrow when needed (just like whole numbers)</li>
    <li>Place the decimal point in the answer directly below the decimal points</li>
  </ol>
</div>

### Worked Example 3: Simple Subtraction

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$8.7 - 3.4$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Write vertically with decimal points aligned
```
  8.7
- 3.4
------
```

**Step 2:** Subtract column by column
- Tenths: 7 - 4 = 3 → write 3
- Ones: 8 - 3 = 5 → write 5

**Step 3:** Place decimal point
```
  8.7
- 3.4
------
  5.3
```

**Answer:** 5.3

---

### Worked Example 4: Subtraction with Borrowing

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$5.23 - 2.47$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Write vertically with decimal points aligned
```
  5.23
- 2.47
------
```

**Step 2:** Subtract column by column (borrow when needed)
- Hundredths: 3 < 7, so borrow from tenths
  - 13 - 7 = 6 → write 6
- Tenths: After borrowing, we have 1 (not 2). 1 < 4, so borrow from ones
  - 11 - 4 = 7 → write 7
- Ones: After borrowing, we have 4 (not 5)
  - 4 - 2 = 2 → write 2

**Step 3:** Place decimal point
```
  5.23
- 2.47
------
  2.76
```

**Answer:** 2.76

### Worked Example 5: Different Decimal Places

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h4 style="color: #1976d2; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$12.5 - 8.347$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Write vertically with decimal points aligned. Add zeros to make both numbers have 3 decimal places
```
 12.500
- 8.347
-------
```

**Step 2:** Subtract column by column
- Thousandths: 0 < 7, borrow from hundredths
  - 10 - 7 = 3 → write 3
- Hundredths: After borrowing, we have 9 (not 10). 9 < 4? No, 9 > 4
  - 9 - 4 = 5 → write 5
- Tenths: 4 < 3? No, 4 > 3
  - 4 - 3 = 1 → write 1
- Ones: 2 < 8, borrow from tens
  - 12 - 8 = 4 → write 4
- Tens: After borrowing, we have 0 (not 1), so nothing to write

**Step 3:** Place decimal point
```
 12.500
- 8.347
-------
  4.153
```

**Answer:** 4.153

## 4) Multiplying Decimals

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">
  <h3 style="color: #1976d2; margin-top: 0;">🔵 Fundamental Principle: Count Decimal Places</h3>
  <p style="margin: 10px 0;"><strong>Rule:</strong> Multiply as if they were whole numbers, then count the total number of decimal places in both factors. Place that many decimal places in your answer.</p>
  <p style="margin: 10px 0 0 0;"><strong>Key steps:</strong></p>
  <ol style="margin: 10px 0 0 0; line-height: 1.8;">
    <li>Multiply the numbers as if they were whole numbers (ignore the decimal points)</li>
    <li>Count the total number of decimal places in both factors</li>
    <li>Place the decimal point in the answer so it has that many decimal places</li>
    <li>Count from right to left when placing the decimal point</li>
  </ol>
  <p style="margin: 10px 0 0 0;"><strong>Memory tip:</strong> "Multiply like whole numbers, then count decimal places."</p>
</div>

### Worked Example 6: Simple Multiplication

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">
  <h4 style="color: #e65100; margin-top: 0;">Problem</h4>
  <p style="margin: 10px 0; font-size: 1.1em; text-align: center;">$$2.3 \times 1.5$$</p>
</div>

**Step-by-step (showing work):**

**Step 1:** Multiply as whole numbers (ignore decimal points)
```
  23
× 15
----
 115
 23
----
 345
```

**Step 2:** Count decimal places
- 2.3 has 1 decimal place
- 1.5 has 1 decimal place
- Total: 1 + 1 = 2 decimal places

**Step 3:** Place decimal point in answer (2 places from right)
```
345 → 3.45
```

**Answer:** 3.45

### Worked Example 8: Multiplying by Powers of 10

<div style="background: #fff3e0; padding: 16px; border-radius: 10px; border-left: 6px solid #ff9800; margin: 18px 0;">

## 5) Dividing Decimals

<div style="background: #f3e5f5; padding: 16px; border-radius: 10px; border-left: 6px solid #9c27b0; margin: 18px 0;">
  <h3 style="color: #7b1fa2; margin-top: 0;">🟣 Fundamental Principle: Move the Decimal Point</h3>
  <p style="margin: 10px 0;"><strong>Rule:</strong> Make the divisor a whole number by moving the decimal point. Move the decimal point in the dividend the same number of places. Then divide as usual.</p>
  <p style="margin: 10px 0 0 0;"><strong>Key steps:</strong></p>
  <ol style="margin: 10px 0 0 0; line-height: 1.8;">
    <li>If the divisor is a decimal, move its decimal point to make it a whole number</li>
    <li>Move the decimal point in the dividend the same number of places</li>
    <li>Divide as if both were whole numbers</li>
    <li>Place the decimal point in the quotient directly above where it is in the dividend</li>
    <li>Add zeros to the dividend if needed to continue dividing</li>
  </ol>
  <p style="margin: 10px 0 0 0;"><strong>Memory tip:</strong> "Move the decimal to make the divisor whole, then move the dividend's decimal the same way."</p>
</div>

### Worked Example 9: Simple Division

<div style="background: #e3f2fd; padding: 16px; border-radius: 10px; border-left: 6px solid #2196f3; margin: 18px 0;">

</div>

**Step-by-step (showing work):**

**Step 1:** The divisor (2) is already a whole number, so no need to move decimal points

**Step 2:** Divide as usual, placing decimal point above dividend's decimal point

  ---

## Summary: Key Rules to Remember

<div style="background: #fff9c4; padding: 18px; border-radius: 10px; border-left: 6px solid #fbc02d; margin: 18px 0;">
  <h3 style="color: #f57f17; margin-top: 0;">📝 Quick Reference</h3>
  <ul style="line-height: 2;">
    <li><strong>Adding:</strong> Line up decimal points, add column by column</li>
    <li><strong>Subtracting:</strong> Line up decimal points, subtract column by column</li>
    <li><strong>Multiplying:</strong> Multiply as whole numbers, count total decimal places</li>
    <li><strong>Dividing:</strong> Move decimal point to make divisor whole, move dividend's decimal the same way</li>
    <li><strong>Powers of 10:</strong> Multiply → move right; Divide → move left</li>
  </ul>
