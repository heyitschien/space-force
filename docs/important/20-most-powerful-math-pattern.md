# 1️⃣ Distance / Speed / Time

Classic ASVAB question.

**Formula**

d = rt

Example:

> A car travels 60 mph for 3 hours. How far?

Solution:

```
distance = 60 × 3
distance = 180 miles
```

Shortcut:

**distance = rate × time**

---

# 2️⃣ Percentage of a Number

Example:

> What is 20% of 150?

Quick method:

```
20% = 0.20
150 × 0.20 = 30
```

---

# 3️⃣ Percentage Increase / Decrease

Example:

> $50 increased by 20%

```
increase = 50 × 0.20 = 10
new value = 60
```

Shortcut:

```
new value = original × (1 + percent)
```

# 4️⃣ Ratio Problems

# 5️⃣ Proportion Problems

Example:

> 3 pencils cost $6. How much for 10?

```

```

Shortcut:
