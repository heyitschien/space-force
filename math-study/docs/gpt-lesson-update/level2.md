# Lesson 1: Ratios & Proportions (Updated)

Your current version is good but missing one visual concept.

### Add this principle:

```
a:b = c:d
```

Cross multiply:

```

```

Students must **see the structure**.

---

### Updated principles

```
Ratio a:b means a/b
Set proportion: a/b = c/d
Cross multiply: ad = bc
Scale factor: multiply both sides
```

---

### Visual Example (Add)

Example:

```
3 red : 4 blue
```

Total blue = 80

Scale factor:

```
80 ÷ 4 = 20
```

Then

```
3 × 20 = 60 red
```

This **scale thinking** is faster than cross multiplication.

---

### Add ASVAB shortcut rule

```

```

This saves huge time.

# Lesson 3: Averages (Major Upgrade)

Your averages lesson is already good, but missing the **ASVAB trick**.

ASVAB averages usually involve **missing numbers**.

Your principle:

```
average = sum ÷ count
```

Good.

Add:

```
sum = average × count
```

---

### Add weighted average concept

Example:

```
4 tests average 85
5th test needed for 88 average
```

Students should think:

```
target sum = average × count
```

Example:

```
88 × 5 = 440
```

Current sum:

```
85 + 90 + 88 + 87 = 350
```

Required score:

```
440 − 350 = 90
```

This is **much faster than algebra**.

Add shortcut rule:

```
target sum − current sum = missing value
```
