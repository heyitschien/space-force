# The Correct Quiz Architecture (Used in Elite Test Prep)

You need **three types of quizzes**:

```
1️⃣ Pattern Mastery Quiz
2️⃣ Speed Drill Quiz
3️⃣ Mixed ASVAB Simulation
```

Each serves a different learning purpose.

---

# 1️⃣ Pattern Mastery Quiz (Per Topic)

These are **individual quizzes for each category**.

Example topics from your Level 1:

* PEMDAS
* Decimal Operations
* Fractions
* Percents
* Unit Conversion

Each topic should have a **15-question mastery quiz**.

Your idea is actually **very close to optimal**.

### Structure

```
15 Questions Total
```

Difficulty curve:

| Difficulty | Count |
| ---------- | ----- |
| Easy       | 5     |
| Medium     | 7     |
| Hard       | 3     |

This is **perfect cognitive load**.

---

### Example: Fractions Quiz

**Easy**

```
3/4 of 24
```

**Medium**

```
Recipe needs 2/3 cup sugar.
How many batches from 5 cups?
```

**Hard**

```
A number is 3/5 of another.
If their sum is 64, find the numbers.
```

These mimic **ASVAB wording progression**.

---

# 2️⃣ Speed Drill Quiz

After the mastery quiz, students should unlock a **Speed Drill**.

```
10 questions
60 seconds
```

Goal:

```
instant recognition
```

Example:

```
20% of 150
0.6 × 0.3
3/5 of 50
```

These should be **very fast problems**.

The purpose is building:

```
calculation muscle memory
```

---

# 3️⃣ Mixed ASVAB Simulation

Once all Level 1 topics are completed, students take a **mixed quiz**.

Example:

```
20 questions
mixed categories
```

Example distribution:

| Topic           | Count |
| --------------- | ----- |
| PEMDAS          | 4     |
| Decimals        | 4     |
| Fractions       | 4     |
| Percents        | 4     |
| Unit Conversion | 4     |

Now the brain learns:

```
pattern recognition
```

which is what the ASVAB actually tests.

# Difficulty Scaling (Very Important)

To make the real ASVAB feel easy, the **hard questions in your app should be harder than the test**.

ASVAB Level vs App Level:

| App Level | ASVAB Equivalent     |
| --------- | -------------------- |
| Easy      | Very Easy            |
| Medium    | Medium               |
| Hard      | Hard                 |
| Expert    | Hardest AR questions |

Add **2 “Expert” questions** to each quiz.

Example:

```
Fraction + Percent combination
```

This makes the real test feel comfortable.
